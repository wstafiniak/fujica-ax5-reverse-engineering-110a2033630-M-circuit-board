
# Fujica AX-5 ‚Äì Bottom PCB Reverse Engineering

This project documents the **reverse engineering of the bottom PCB from the Fujica AX-5 camera**.  
The work focuses on analyzing the original electronic layout, identifying components and
connections, recreating the schematic, and designing a **new replacement PCB** using EasyEDA.

The project is intended for **educational and documentation purposes**, aimed at electronics
enthusiasts, analog camera technicians, and anyone interested in preserving and repairing
vintage photographic equipment.

---

## Project goals

- Analyze and understand the original Fujica AX-5 electronic circuit  
- Identify components and PCB trace connections  
- Recreate the electrical schematic with referenced parts  
- Design a new PCB based on the original layout  
- Preserve technical knowledge for future repair and restoration work
